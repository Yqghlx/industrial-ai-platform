# 工业AI平台产品手册

---

## 🏭 产品概述

工业AI智能监控平台是面向工业设备预测性维护的智能解决方案，集成AI大模型分析、实时数据监控、告警预警于一体。

---

## ✨ 核心功能

### 1. 实时设备监控
- **设备状态追踪**：实时显示64+台设备运行状态
- **遥测数据采集**：温度、振动、压力、功耗等指标
- **可视化Dashboard**：设备地图、实时图表、状态指示

### 2. AI智能分析
- **自然语言查询**："这台设备温度异常，帮我分析原因"
- **故障预测**：基于历史数据预测潜在故障
- **维护建议**：自动生成维护计划和建议

### 3. 智能告警系统
- **规则引擎**：温度>80°C、振动>3.5触发告警
- **告警分级**：P0紧急、P1重要、P2一般
- **自动推送**：飞书/钉钉/邮件多渠道通知

### 4. ROI统计分析
- **故障率统计**：设备故障率趋势分析
- **维护成本**：预测性维护vs事后维护成本对比
- **投资回报**：ROI计算器（节省成本/投入成本）

## 🚀 快速开始

### 部署方式
1. **SaaS订阅**：开通账号即可使用
2. **私有部署**：Docker/Kubernetes一键部署
3. **混合部署**：核心数据本地，AI云端

### 接入步骤
1. 安装Edge SDK（C#/Python/Go）
2. 配置设备上报规则
3. 设置告警阈值
4. 开始AI智能分析
