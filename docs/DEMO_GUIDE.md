## 📋 Demo流程（10分钟）

### 1. Dashboard概览（2分钟）

**展示内容**：
- 设备地图：显示64台设备分布
- 实时状态：正常/告警/故障设备统计
- 遥测曲线：温度/振动实时图表

**讲解要点**：
- WebSocket实时推送，数据延迟<1秒
- TimescaleDB时序存储，自动压缩节省90%空间

### 2. 设备详情查看（2分钟）

**操作步骤**：
1. 点击设备地图中任意设备
2. 查看设备详情页（温度历史、告警记录）
3. 切换时间范围（1小时/1天/1周）

**讲解要点**：
- 响应时间<50ms，前端React优化
- 历史数据查询支持百万级记录

### 3. AI智能分析演示（3分钟）

**操作步骤**：
1. 打开AI对话界面
2. 输入查询："设备001温度为什么异常？"
3. 等待AI回复（约5-25秒）
4. 展示AI分析结果和建议

**讲解要点**：
- GLM-4-Flash大模型，30秒响应
- LLM队列优化，支持10并发
- 答案缓存，常见问题秒回

### 4. 告警触发演示（1分钟）

**操作步骤**：
1. 模拟设备温度超80°C
2. 观察告警自动触发
3. 查看飞书/钉钉推送（配置后）

**讲解要点**：
- 规则引擎实时检测，延迟<500ms
- 多渠道推送，告警分级管理

### 5. ROI统计分析（2分钟）

**操作步骤**：
1. 打开ROI统计页面
2. 查看故障率趋势图
3. 展示投资回报计算

**讲解要点**：
- 自动统计设备故障率
- ROI计算器量化价值
