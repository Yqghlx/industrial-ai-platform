## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 边缘层 | C# .NET 8 | 设备数据模拟器 |
| 后端层 | Go 1.25 + Gin | RESTful API + WebSocket |
| 数据层 | PostgreSQL + TimescaleDB | 时序数据存储 + 自动压缩 |
| AI层 | 百炼 GLM-5 | 大语言模型 + 规则引擎回退 |
| 前端层 | React 19 + TypeScript | 组件化 UI + 严格类型检查 |
| 实时层 | WebSocket | 双向通信 + 心跳 + 数据压缩 |
| 缓存层 | Redis/Memory | 可选缓存层 |
| CI/CD | GitHub Actions | 自动化测试 + Docker构建 |

## 数据流

### 1. 设备数据摄入
```
Edge Simulator → Backend API → PostgreSQL (TimescaleDB)
```
- C# 模拟器生成设备遥测数据
- HTTP POST `/api/v1/devices/telemetry`
- TimescaleDB 自动压缩存储

### 2. 实时数据推送
```
Backend → WebSocket → Frontend
```
- WebSocket 双向通信
- 心跳保活机制
- 数据压缩节省带宽

### 3. AI 查询流程
```
Frontend → Backend API → AI Agent → GLM-5
```
- 用户输入自然语言查询
- AI Agent 路由到对应专家
- GLM-5 返回智能分析结果

### 4. 时序数据存储
```
Backend → PostgreSQL → TimescaleDB
```
- 按时间自动分区
- 7天前数据自动压缩
- 90天前数据自动删除
